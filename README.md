# RemindMeBot

Простой чат-бот, который напоминает команде проекта о командных 
мероприятиях, событиях, регулярных встречах и т.п.

## Структура проекта

- `cmd/` - Этот каталог содержит точки входа для наших приложений, которые 
мы собираем в контейнеры Docker. В нашем случае это `bot` и `reminder`.
- `pkg/` - Каталог `pkg` содержит наш общий код, разделенный по пакетам. В 
этом случае у нас есть пакеты `bot`, `reminder`, `postgres`, `nats`.
- `docker-compose.yml` - файл, описывающий нашу конфигурацию Docker 
Compose, который мы используем для развертывания и запуска нашего 
приложения.
- `Makefile` - это файл с инструкциями для автоматизации различных задач, 
таких как сборка приложений, создание баз данных и т.д.
- `README.md` - файл, содержащий инструкции по установке, настройке и 
использованию приложения.
